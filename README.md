# 基于Servlet+jsp的商城系统

### 介绍
51商城采用JavaWeb技术开发的一个B2C的在线商城系统,提供商品购买,下单,购物车以及后台数据管理功能; 适用于大作业或毕业设计

### 软件架构

语言平台: JavaEE,JDK1.8
数据库: SQLServer2008/2014
服务器: Tomcat8
开发工具: IntelliJ IDEA 2019.3.1
技术要点: Servlet, JSP, JDBC
